Machine Learning

Tunesight

Desenvolvimento de aplicação de Machine Learning utilizando Python, Pandas, NumPy e Scikit-Learn em base com 7.746 amostras reais de músicas extraídas de múltiplas fontes, aplicando análise exploratória de dados, tratamento de outliers por intervalo interquartil (IQR), balanceamento de classes, cross-validation, engenharia de atributos (extração de 17 features de áudio), seleção de hiperparâmetros e desenvolvimento de uma integração via API Flask para a comunicação automatizada entre o modelo de classificação e uma interface de chatbot. O modelo preditivo final alcançou 78% de acurácia balanceada em cenários com dados inéditos de teste

Similaridade de Reviews com NLP

Desenvolvimento de aplicação de Processamento de Linguagem Natural (NLP) utilizando Python, Pandas, Scikit-Learn, Sentence Transformers e Gradio para identificação de similaridade textual em reviews. O projeto envolveu pré-processamento de textos, vetorização com TF-IDF, comparação semântica por embeddings, cálculo de similaridade por cosseno, avaliação comparativa de diferentes abordagens de NLP e implementação de interface interativa para consulta e recuperação automática das avaliações mais semelhantes a partir de textos fornecidos pelo usuário.

Análise de Dados

Análise Black Friday: Zoop Megastore

Desenvolvimento de projeto de Análise de Dados utilizando SQL, Python e Excel em base relacional composta por mais de 25 mil registros distribuídos entre clientes, vendas, produtos, categorias, marcas e fornecedores. Aplicação de análise exploratória de dados (EDA), construção de KPIs de faturamento, ticket médio, volume de vendas e segmentação de clientes, utilizando consultas SQL avançadas com agregações e CTEs para identificar padrões de consumo e oportunidades de crescimento. A análise revelou que a Black Friday representa aproximadamente 18% das vendas anuais da empresa e gera um volume de pedidos até 2,4 vezes superior à média mensal, resultando na elaboração de recomendações estratégicas para marketing, gestão de estoque, relacionamento com fornecedores e planejamento operacional.

Análise Histórica dos Avengers: Evolução da Equipe, Representatividade e Protagonismoo

Desenvolvimento de projeto de Análise Exploratória de Dados (EDA) utilizando Python, Pandas, NumPy, Matplotlib e Seaborn. Aplicação de limpeza e validação de dados, tratamento de inconsistências temporais, análise estatística descritiva, construção de métricas de representatividade, mortalidade e protagonismo, além do desenvolvimento de visualizações para análise temporal e comparação entre grupos. A análise identificou padrões de expansão da equipe ao longo de 50 anos, evolução da representatividade de gênero, concentração de protagonismo em um pequeno grupo de personagens e relação entre relevância narrativa, mortes e retornos
